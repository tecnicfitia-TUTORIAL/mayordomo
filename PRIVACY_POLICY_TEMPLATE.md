# Política de Privacidad para Confort 65/35

**Última actualización:** [Fecha Actual]

Confort Technologies Inc. ("nosotros", "nuestro") opera la aplicación móvil Confort OS (el "Servicio"). Esta página le informa sobre nuestras políticas con respecto a la recopilación, el uso y la divulgación de datos personales cuando utiliza nuestro Servicio.

## 1. Interpretación y Definiciones
El "Sistema 65/35" se refiere a la lógica operativa de la aplicación donde la IA gestiona datos para reducir la carga cognitiva del usuario.

## 2. Recopilación y Uso de Datos

### Tipos de Datos Recopilados
Para proporcionar el Servicio, podemos solicitarle permisos para acceder a:
*   **Entrada de Audio (Micrófono):** Utilizado exclusivamente para comandos de voz y notas rápidas. El procesamiento se realiza en tiempo real y no se almacena audio crudo permanentemente.
*   **Archivos y Medios (Cámara/Galería):** Utilizado para que el usuario suba facturas, documentos o capturas de pantalla para su análisis.
*   **Ubicación (Opcional):** Utilizada para sugerencias contextuales de logística y clima.

### Uso de Inteligencia Artificial (Google Gemini)
Nuestro Servicio utiliza tecnología de IA generativa para procesar sus datos.
*   Los datos enviados a la IA (texto de chat, contenido de documentos) son **anonimizados** cuando es posible.
*   No utilizamos sus datos personales para entrenar modelos públicos de IA sin su consentimiento explícito.

## 3. Retención de Datos
Conservaremos sus Datos Personales solo durante el tiempo que sea necesario para los fines establecidos en esta Política de Privacidad.

## 4. Eliminación de Datos (Derecho al Olvido)
Usted tiene derecho a solicitar la eliminación completa de su cuenta y todos los datos asociados.
*   **En la App:** Vaya a Ajustes > Zona de Peligro > Eliminar Cuenta.
*   Esta acción es irreversible y elimina sus patrones de comportamiento, historial de chat y configuración de nuestra base de datos.

## 5. Seguridad de los Datos
La seguridad de sus datos es importante para nosotros, pero recuerde que ningún método de transmisión por Internet o método de almacenamiento electrónico es 100% seguro. Nos esforzamos por utilizar medios comercialmente aceptables (Encriptación TLS, Bases de Datos Seguras) para proteger sus Datos Personales.
